# SEE CHIA WAI Portfolio

Static personal portfolio for GitHub Pages.

## Local Preview

```bash
python3 -m http.server 4176
```

Open `http://127.0.0.1:4176`.

## Recommended GitHub Pages Setup

Use a user site repository if the username is available:

```text
seechiawai.github.io
```

GitHub Pages URL:

```text
https://seechiawai.github.io
```

If the username is different, create a repository named:

```text
<github-username>.github.io
```

Then upload these files to the repository root and enable GitHub Pages from the
main branch root.
